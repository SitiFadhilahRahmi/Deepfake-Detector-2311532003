Nama Proyek IsItFake — Deepfake Detection System
Nama Mahasiswa Siti Fadhilah Rahmi

Deskripsi Sistem deteksi deepfake berbasis deep learning menggunakan arsitektur EfficientNetB0, dilatih untuk membedakan wajah manusia asli dengan wajah hasil generate atau dimanipulasi menggunakan AI secara otomatis.
🧠
Deep Learning
Dibangun di atas EfficientNetB0 dengan transfer learning dari ImageNet.
🎯
Akurasi Tinggi
Mencapai akurasi 98% pada validation set dengan 4.000 sampel uji.
⚡
Inferensi Cepat
Hasil deteksi tersedia dalam hitungan detik langsung di browser.
⚠️
Gunakan dengan Bijak
Model ini adalah alat bantu, bukan keputusan akhir.
ℹ️ Alat ini akurat namun tidak sempurna. Gunakan hasilnya sebagai indikator kuat, bukan keputusan akhir. Selalu pertimbangkan konteks di luar hasil deteksi.
